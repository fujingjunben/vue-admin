# antd-demo

## 目录说明
### api
所有接口方法放到api下面的api.js文件

### net
网络请求相关的类放到这里

### keys
常量放到这里

### utils
工具类放到这里

### store
全局数据放在这里

## 代码规范
1. 文件名驼峰命名
2. 所有数据从Api.js中获取
